About

Native tools, one React layer.

I'm a Senior React Native Developer currently building at Brilliant Gamez, where I take apps through full release cycles — Xcode and Android Studio builds, provisioning profiles, signing certificates, and Gradle configuration included.

Most of my recent work sits at the boundary between the React layer and the native one. I've integrated AI/LLM features that stream responses inside a mobile UI — a specific kind of async, connection-drop-prone edge case most React Native devs haven't had to solve yet. I also bridge custom Objective‑C and Java modules, wire up Adapty for subscriptions, and get AppLovin MAX and AdMob mediation to behave.

Before that, at CODENAX and Balawal IT House, I worked across Firebase‑backed products — auth, Firestore, social login, analytics, crash reporting — and shipped four‑plus production apps with reusable component libraries built to accelerate the whole team, not just my own tickets.
